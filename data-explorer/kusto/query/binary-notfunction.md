# <a name="binary_not"></a>binary_not()

Devuelve una negación bit a bit del valor de entrada.

```kusto
binary_not(x)
```

**Sintaxis**

`binary_not(`*num1*`)`

**Argumentos**

* *num1*: numérico 

**Devuelve**

Devuelve la operación lógica NOT en un número: num1.
