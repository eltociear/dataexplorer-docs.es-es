# <a name="gamma"></a>gamma()

Calcula la [función gamma](https://en.wikipedia.org/wiki/Gamma_function)

**Sintaxis**

`gamma(`*X*`)`

**Argumentos**

* *x*: Parámetro para la función gamma

**Devuelve**

* Función gamma de x.
* Para calcular la función log-gamma, véase [loggamma()](loggammafunction.md).
