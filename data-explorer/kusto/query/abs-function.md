# <a name="abs"></a>abs()

Calcula el valor absoluto de la entrada.  

**Sintaxis**

`abs(`*X*`)`

**Argumentos**

* *x*: Un número entero o real, o un valor de intervalo de tiempo.

**Devuelve**

* Valor absoluto de x.
