# <a name="pow"></a>pow()

Devuelve el resultado de elevar a la energía

**Sintaxis**

`pow(`*base*`,` *exponente* base`)`

**Argumentos**

* *base*: Valor base.
* *exponente*: Valor exponente.

**Devuelve**

Devuelve la base elevada al exponente de potencia: base - exponente.
